## ğŸš€ Template yang Tersedia

### 1. Vite + TypeScript (`vite-ts/`)
**Fitur:**
- Dev server sangat cepat (ESM-native)
- Hot Module Replacement (HMR)
- Build ringan dan cepat

**Menjalankan project:**
```
cd vite-ts
npm install
npm run dev
```



### 2. Webpack + TypeScript (`webpack-ts/`)
**Fitur:**
- Konfigurasi bundler fleksibel
- Menggunakan `ts-loader` untuk kompilasi TypeScript

**Menjalankan project:**
```
cd webpack-ts
npm install
npm run dev
```



### 3. ESBuild + TypeScript (`esbuild-ts/`)
**Fitur:**
- Bundling super cepat
- Setup minimal
- Cocok untuk proyek kecil & landing page

**Menjalankan project:**
```
cd esbuild-ts
npm install
npm run dev
```



### 4. Native ES Modules + TypeScript (`esmodule-ts/`)
**Fitur:**
- Tanpa bundler
- Output sangat bersih
- Cocok untuk demo, eksperimen, atau modul kecil

**Menjalankan project:**
```
cd esmodule-ts
npm install
npm run build
npm start
```
